# Medicine QR Generator 🏥

#Project Description
Medicine QR Generator is a web-based application that generates and scans QR codes for medicines. 
It extracts medicine information using CSV and displays usage details.

## 🚀 Features
- Generate QR codes for medicines
- Scan and recognize medicine names
- Fetch details from dataset
- Display usage, side effects, and instructions
- Audio output support

## 🛠 Technologies Used
- Python
- Flask
- Pandas
- HTML & CSS

## 📂 Project Structure
- app.py → Main application file
- templates/ → HTML files
- static/ → Images, QR codes, audio
- dataset.csv → Medicine database

## 🎯 Future Improvements
- Cloud deployment
- AI-based medicine recognition
- Mobile app version
